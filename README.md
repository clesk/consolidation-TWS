### Overview
This repository contains code and data to reproduce the results in Lesk & Mankin, 2026. 


### Workflow overview

The input datasets for the study are listed in Table S1. They are first processed into derived datasets (summarized in Fig. 1), which are then assembled into the panels used for the statistical models (Figs. 2-3). Those same input datasets are used to run the process-based models (Fig. 4). The historical and projected impacts are estimated from the results of the statistical models combined with input and derived data (Fig. 5).
